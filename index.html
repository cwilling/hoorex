HooRex

hoorex is a small utility program which uses information from SlackBuilds.org (SBo) .info files to calculate various package dependency relationships. In particular, it shows Who Requires (which other packages require) a particular package (or a number of packages) in the SBo repository.

HooRex overview

HooRex is a dependency calculator. It uses information from the REQUIRES field of the .info file for each package available in the SlackBuilds.org (SBo) repository. The initial motivation for creating HooRex was to know which other packages might be affected by a change to any particular package e.g. if a new version of ffmpeg is built and installed, which other packages, by virtue of their dependence on ffmpeg, might be affected - posssibly requiring to be rebuilt themselves? Of course it is possible to find all the .info files and grep for REQUIRES in them but the result is not clean enough, as is, for passing on to another program or script e.g. a build bot. Also its somewhat slow to do this (there are over 5000 packages in the SBo repo), especially when dealing with multiple packages. This (multiple packages to check) is almost always the situation after an update to the SBo repo when I want to know, programmatically, not just which of the updated packages to rebuild but also which other packages depend on the updated packages and so may need rebuilding as well. HooRex enables this quickly and painlessly.
